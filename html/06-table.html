Tablas HTML

¿Qué es una tabla?

Una tabla es un conjunto estructurado de datos distribuidos en filas y columnas (datos tabulados). Una tabla permite buscar con rapidez y facilidad valores entre diferentes tipo de datos que indiquen algún tipo de conexión. Por ejemplo, una persona y su edad, o un día de la semana o el jorario de una piscina municipal.

Etiqueta de tabla HTML

La etiqueta <table> se utiliza para crear tablas en páginas web. Organiza los datos en filas y columnas, de forma similar a una hoja de calculo. Dentro de la etiqueta <table>, encontrarás etiquetas como <tr> para filas de datos, <th> para los encabezados de la tabla y <td> para las celdas de datos de la tabla, que definen la estructura y el contenido de la misma. 

            
              <table>

                <tr>
                  <th>Country</th>
                  <th>Capital</th>
                  <th>Population</th>
                </tr>

                <tr>
                  <td>Germany</td>
                  <td>Berlin</td>
                  <td>83 million</td>
                </tr>

                <tr>
                  <td>France</td>
                  <td>Paris</td>
                  <td>65 million</td>
                </tr>

                <tr>
                  <td>Japan</td>
                  <td>Tokyo</td>
                  <td>126 million</td>
                </tr>

              </table>
