Tu primer archivo HTML

Antes de empezar

Sé que lo que queremos es empezar a crear, pero siempre es importante tener los conceptos bases y las reglas de cualquier cosa que vallamos a utilizar, en este caso HTML, si tenemos claro que podemos hacer y que no, las cosas serán más fácil cuando creemos nuestro primer archivo HTML.

Etiquetas y Atributos

Cuando hablamos anteriormente de los elementos HTML mencionamos que se componen de una etiqueta de inicio, el contenido y la etiqueta de cierre, entonces ya sabemos que las etiquetas son las que se encarga de definir los elementos, por ejemplo <h1>Ejemplo</h1>, como podemos ver las etiquetas tienen un nombre en este caso h1 que para el navegador significa que ese texto es un título.

Entones sabiendo lo que son las etiquetas podemos hablar de los atributos. Los atributos nos permiten definir información extra o propiedades adicionales a las etiquetas, por ejemplo en el caso de una imagen <img src="ejemplo.jpg" alt="Ejemplo"> el atributo src nos indica la ruta de la imagen y el atributo alt nos indica el texto alternativo de la imagen.

Insensibilidad a mayúsculas y minúsculas

Realmente por buenas prácticas las etiquetas HTML la escribimos en minúsculas, pero realmente HTML no diferencia entre mayúsculas y minúsculas, por lo tanto no importa si escribimos.

<p>Ejemplo</p> <P>Ejemplo</P> <p>Ejemplo</P> <P>Ejemplo</p>

Entidad

Una entidad es un conjunto de caracteres (string) especiales que se utilizan para imprimir en pantalla caracteres reservados en este caso por HTML o caracteres invisibles (tabulaciones o espacios en blanco). También pueden usarse para representar caracteres que no existen en algun teclado, como por ejemplo la Ñ o acentos. Un error muy común cuando se aprende HTML es no entender.

Caracteres Reservados

Los caracteres reservados en HTML son caracteres que tienen un significado especial en HTML y por lo tanto no se pueden usar directamente en el contenido de una página web.

Algunos ejemplo comunes:

Caracter Entidad Nota
< &lt; Es usado para abrir una etiqueta HTML.
> &gt; Es usado para cerrar una etiqueta HTML.
& & Se usa para iniciar una entidad HTML.
" &quot; Usado para valores de atributos.
' &apos; Usado para valores de atributos.

Comentarios

Los comentarios en HTML son notas que puedes agregar a tu código para explicarlo, hacer recordatorios o desactivar temporalmente código. Los comentarios no se muestran en el navegado solo son visible en el código fuente. Para crear un comentario se utiliza la siguiente sintaxis: 

            
              <!-- Este es un comentario -->
              <p>Este es un parrafo</p>

En el navegador solo se vera:

            
Este es un parrafo
