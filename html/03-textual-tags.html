Etiquetas textuales

Las etiquetas textuales se utilizan para estructuras y dar formato al contenido de texto en una página web. Normalmente con un sentido semántico y otras veces simplemente darle un toque a lo visual (Recomiendo que siempre se le de un sentido semántico, dejemo el diseño para CSS).

Encabezados

Los encabezados se utilizan para definir los títulos y subtítulos de un documento. HTML ofrece seis niveles de encabezados, desde <h1> el más importante hasta <h6> el menos importante. Estos ayudan a estructurar el contenido y mejorar la legibilidad al crear una jerarquía clara.

Etiqueta de título

La etiqueta <title> tiene bastante importancia para la identificación de la páginas en internet, este elemento se encarga de darle titulo a una página web, este título se muestra en la la pestaña del navegador y en los resultados de los motores de búsquedas.

Párrafos

Esta etiqueta le indica al nevegador que todo el texto dentro de si es un párrafo, aunque visualmente no sea diferente a otros textos semánticamente es distinto. <p>
