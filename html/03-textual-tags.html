Etiquetas textuales

Las etiquetas textuales se utilizan para estructuras y dar formato al contenido de texto en una página web. Normalmente con un sentido semántico y otras veces simplemente darle un toque a lo visual (Recomiendo que siempre se le de un sentido semántico, dejemo el diseño para CSS).

Encabezados

Los encabezados se utilizan para definir los títulos y subtítulos de un documento. HTML ofrece seis niveles de encabezados, desde <h1> el más importante hasta <h6> el menos importante. Estos ayudan a estructurar el contenido y mejorar la legibilidad al crear una jerarquía clara.

Etiqueta de título

La etiqueta <title> tiene bastante importancia para la identificación de la páginas en internet, este elemento se encarga de darle titulo a una página web, este título se muestra en la la pestaña del navegador y en los resultados de los motores de búsquedas.

Párrafos

Esta etiqueta le indica al nevegador que todo el texto dentro de si es un párrafo, aunque visualmente no sea diferente a otros textos semánticamente es distinto. <p>

Saltos de línea

En el caso de la etiqueta <hr/> nos ayuda a crear una línea horizontal que separa el contenido, indicando un cambio temático.

Por otro lado, la etiqueta <br/> nos ayuda a crear un salto de línea, por lo generar dentro la misma etiqueta <p>.

b / strong

Visualmente <b> y <strong> hacen lo mismo (color o tamaño de negrita), pero no semánticamente, la etiqueta <strong> indica que el texto es de suma importancia.

pre

La <pre> etiqueta en HTML representa texto preformateado. El texto dentro de un <pre> element se muestra con una fuente de ancho fijo y conserva tanto los espacios como los saltos de línea. Esto resulta útil para mostrar fragmentos de código, arte ASCII o cualquier otro texto donde el formato sea importante, osea le da importancia semántica al texto. 

            
              function greet(name) {

i / em

Por un lado tenemos la etiqueta <i> que representa texto en cursiva, por otro lado tenemos la etiqueta <em> que representa texto en cursiva con énfasis o semánticamente hablando.

mark

La etiqueta <mark> en HTML se utiliza para resaltar o marcar un texto que tiene especial importancia o relevancia en el contexto de un documento. El texto dentro de un <mark> se suele mostrar con un color de fondo amarillo para indicar su destaque.
