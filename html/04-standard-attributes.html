Atributos estándar

Los atributos estándar son propiedades que se pueden usar en casi todos los elementos HTML para proporcionar información o funcionalidad adicional. Estos atributos controlan aspectos como el estilom la identificación, la accesibilidad y el manejo de eventos de los elementos, lo que permite modificar su comportamiento e interacción dentro de una página web. Ofrecen una forma consistente de gestionar las características comunes de los elementos en diferentes navegadores y dispositivos.

id

El atributo global id define un identificador único (ID) el cual no debe repetirse en todo el documento. Su propósito es identificar el elemento al vincularlo (usando un identificador de fragmento), en scripts u hojas de estilo (con CSS).

  • Debe ser único
  • No debe contener espacios en blanco
  • No debe de empezar con número ni simbolos especiales

class

El atributo global class es una lista de las clases del elemento separada por espacios. Las clases permiten a CSS y Javascript seleccionar y acceder a elementos específicos a través de los selectores de clases o funciones como el método document.getElementsByClassName del DOM.

  • Usar nombre de clases, que describan el propósito semántico del elemento

Uso de atributos de datos

HTML5 está diseñado de forma tal que sea fácil extender los datos asociados a un elemento en particular sin necesidad de que tengan un significado definido. Los atributos data-* permiten almacenar indormación adicional sobre un elemento HTML cualquiera.

Sintaxis HTML

La sintáxis es simple. Un atributo cualquiera cuyo nombre comience con data- es un atributo de datos. Suponganmos que tenemos un artículo y deseamos almacenar información adicional que no tiene ninguna representación visual: 

            
              <article
                id="electriccars"
                data-columns="3"
                data-index-number="12314"
                data-parent="cars"
              >
                ...
              </article>

Ejemplo de como acceder con Javascript

            
              let article = document.getElementById("electriccars");

              article.dataset.columns; // "3"
              article.dataset.indexNumber; // "12314"
              article.dataset.parent; // "cars"

Ejemplo de como acceder con CSS

            
              article[data-columns="3"] {
                width: 400px;
              }
              article[data-columns="4"] {
                width: 600px;
              }

style

Hay un atributo style que sirve para agregar css directamente en el elemento, pero no quiero hablar de el porqué siempre voy a recomendar que los estilos se manejen por separado. No obstante, es bueno saber que existe para casos muy puntuales.
