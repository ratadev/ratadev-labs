Etiquetas básicas de HTML

Existen una etiquetas que si o si tienen que estar en un archivo HTML para que este pueda funcionar, de estas etiquetas hablaremos en esta oportunidad.

Doctype

La declaración <!doctype html> no es una etiqueta, nos podemos referimos a ella como 'document type declaration' o por las siglas 'DTD' y basicamente se encarga de decirle al navegador qué versión de html estamos utilizando.

Elemento HTML

El <html> elemento es la raíz de todo documento HTML. Todos los demás elementos HTML (excepto el Doctype declaración) deben ser descendientes de este elemento. A este elemento se le conoce como elemento raíz o raíz del documento.

Elemento Head

En este elemento es donde se alojan toda la información de la página web, metadatos y enlaces de los archivos, pero nada de lo que vaya dentro de este elemento se mostrara dentro del contenido de la página web. Es muy importante para el SEO (Search Engine Optimization) y para que los navegadores puedan entender el contenido de la página web.

Elemento Body

En el <body> elemento es donde se alojan todos los elementos que se mostrarán dentro del contenido de la página web, textos, imagenes, tablas y todo lo demás.

Metadatos

Las metaetiquetas son elementos HTML que proporcionan metadatos sobre una página web. Estos datos no se muestran en la página en sí, sino que los navegadores, los motores de búsqueda y otros servicios web los utilizan para comprender el contenido y el propósito de la página. Se ubican dentro de la sección <head> de un documento HTML y pueden especificar información como el conjunto de caracteres, la descripción, las palabras clave, el autor y la configuración de la ventana gráfica.
