HTML (HyperText Markup Language)

¿Qué es HTML?

Cuando empecé a aprender HTML muchas veces me tope con la frace "HTML es el componente mas básico de la web", y ahora con unos año de experiencia y varias paginas desarrolladas, esa frace me deja dudas. ¿Cómo se puede decir que es básico, si sin HTML no hay página web?, claro por que si HTML me da la estructura y se encarga de definir cada elemento de la página, o sea, títulos, imagenes, parrafós, etc. Yo supongo que se refieren a que es básico en comparación con lo que podemos hacer con CSS y JavaScript, o es basico por que es rapido de aprender y usar. Pienso que no darle la importancia y el tiempo a aprender HTML es un error que se repite mucho en las personas nuevas en este mundillo, por eso vemos paginas web que son una porqueria a nivel de SEO, semantica y estructura, aunque se vean bonitas por fuera.

Entonces ¿Qué es HTML?, HTML o HyperText Markup Lenguage, es un lenguaje de marcado que se utiliza para crear la estructura y definir cada elemento de una página web, o sea, le decimos al navegador qué es cada cosa y cómo debe mostrarse.

Siguendo la definicion podemos destacar varias palabras claves, que al deglosarlas nos ayuda a tender mejor de que se trata HTML y esas palabras son: HyperText, Lenguaje de marcado e Elementos

HyperText

HyperText o texto hipervinculado, se trata de enlaces que conectan páginas web, ya sean del mismo sitio o entre sitios web.

Elementos

Los elementos se encargan de decirle al nevegador que es cada cosa, es decir, que es un título, que es un párrafo, que es una imagen, etc.

Por ejemplo, el siguiente codigo HTML:

            
              <!DOCTYPE html>
              <html>
                <head>
                  <title>Ejemplo</title>
                </head>
                <body>
                  <h1>Ejemplo</h1>
                  <p>Ejemplo</p>
                  <img src="ejemplo.jpg" alt="Ejemplo">
                </body>
              </html>

nos dice que la pagina tiene un título, un encabezado, un párrafo y una imagen.

Los elemento mayormente se compone de tres parte, la etiqueta de apertura, el contenido y la etiqueta de cierre, por ejemplo <h1>Contenido</h1>. Sin embargo, no todos los elementos tienen etiqueta de cierre, como por ejemplo <img> que es una etiqueta vacia: <img src="ejemplo.jpg" alt="Ejemplo">.

Lenguaje de marcado

Siempre me gusta usar el ejemplo del resaltador para explicar lo que son los lenguajes de marcados. Cuando tenemos que estudiar un documento o entender su contenido. Una buena herramienta es el resaltador, ya que nos ayuda a resaltar las palabras claves, las definiciones, las fechas, etc. y podemos agregarle notas para terminar de entender el contenido. Podemos acercar la lógica del lenguaje de marcado a ese ejemplo, en este caso HTML su resaltado son los elementos HTML, y las etiquetas sería esas notas que define que es cada elemento.
