HTML (HyperText Markup Language)

¿Qué es HTML?

Cuando empecé a aprender HTML muchas veces me tope con la frace "HTML es el componente mas básico de la web", y ahora con unos año de experiencia y varias paginas desarrolladas, esa frace me deja dudas. ¿Cómo se puede decir que es básico, si sin HTML no hay página web?, claro por que si HTML me da la estructura y se encarga de definir cada elemento de la página, o sea, títulos, imagenes, parrafós, etc. Yo supongo que se refieren a que es básico en comparación con lo que podemos hacer con CSS y JavaScript, o es basico por que es rapido de aprender y usar. Pienso que no darle la importancia y el tiempo a aprender HTML es un error que se repite mucho en las personas nuevas en este mundillo, por eso vemos paginas web que son una porqueria a nivel de SEO, semantica y estructura, aunque se vean bonitas por fuera.

Entonces ¿Qué es HTML?, HTML o HyperText Markup Lenguage, es un lenguaje de marcado que se utiliza para crear la estructura y definir cada elemento de una página web, o sea, le decimos al navegador qué es cada cosa y cómo debe mostrarse.
